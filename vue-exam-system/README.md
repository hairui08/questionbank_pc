# 题库系统管理后台 (Vue 3 + TypeScript + Vite)

一个功能完整的题库管理系统,支持多种题型的录入、管理和组织。

## ✨ 功能特性

### 核心功能

- ✅ **项目管理** - 创建和管理考试项目
- ✅ **章节管理** - 科目、章节、小节的三级结构管理
- ✅ **题型管理** - 科目级别的题型配置和显示名称映射
- ✅ **题库管理** - 支持 6 种题型的试题录入和管理
- ✅ **设计规范** - 统一的 UI 规范和技术约束文档

### 题型支持

#### 常规题型
1. **单选题** - 单选答案 + 多个选项(2-10个)
2. **多选题** - 多选答案 + 多个选项(2-10个)
3. **判断题** - 正确/错误二选一
4. **不定项** - 多选答案,至少选择一个
5. **简答题** - 文本答案(最多2000字符)

#### 特殊题型
6. **组合题** - 包含案例背景 + 多个小问
   - 案例背景: 最多5000字符
   - 小问题型: 单选、多选、判断、简答 (**不支持不定项**)
   - 小问数量: 不限,建议3-8个

### 筛选功能

支持多维度筛选定位试题:

- **三级联动**: 项目 → 科目 → 章节
- **试题来源**: 历年真题/模拟试题/自定义
- **所属年份**: ≥2000的整数
- **试题级别**: 基础/中级/高级
- **试题难度**: 简单/中等/困难

### 保存模式

- **保存**: 完全清空表单
- **保存并继续添加**: 保留筛选条件和题型,清空题干/选项/答案/解析

## 🚀 快速开始

### 安装依赖

```

```

```bash
npm run build
```

```bash
npm run preview
```

## 🎯 设计原则

### 用户体验
- **三级联动**: 确保数据归属清晰
- **动态选项**: 客观题支持2-10个选项
- **连续录入**: 提供"保存并继续添加"模式
- **表单验证**: 实时验证,及时反馈

### 技术实现
- **Vue 3 Composition API**: 使用 `<script setup>` 语法
- **TypeScript**: 严格类型检查
- **Pinia**: 状态管理
- **Vue Router**: 路由管理
- **CSS 变量**: 统一设计 tokens

### 性能优化
- **热更新**: Vite 提供快速热更新
- **按需加载**: 路由级别代码分割
- **虚拟滚动**: 支持大量数据展示

## ⚠️ 重要说明

### 组合题限制

组合题的小问**仅支持 4 种题型**:
- ✅ 单选题
- ✅ 多选题
- ✅ 判断题
- ✅ 简答题
- ❌ **不支持不定项**

这是基于原型设计的有意限制,确保组合题的复杂度可控。

### 数据约束

| 字段 | 约束 |
|------|------|
| 题干内容 | 1-5000 字符,同科目同章节唯一 |
| 选项内容 | 1-500 字符/选项 |
| 试题解析 | 1-3000 字符 |
| 案例背景 | 1-5000 字符(组合题) |
| 小问答案 | 1-2000 字符(简答题小问) |

## 🛠️ 技术栈

- **框架**: Vue 3.5.13
- **语言**: TypeScript 5.7.3
- **构建**: Vite 7.1.10
- **状态管理**: Pinia 2.3.1
